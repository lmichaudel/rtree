# R-Tree

Sources en C++ de mon TIPE d'informatique où j'implémente la structure de [R-Arbre](https://fr.wikipedia.org/wiki/R-arbre) [1].

Voir la [MCOT](mcot.pdf) pour une brève explication de ce que sont les R-Arbres.
